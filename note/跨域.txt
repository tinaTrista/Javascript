使用xhr(XMLHttpRequest)实现Ajax通信的一个主要限制是只能访问同一个域中的资源
<1> 怎么样才算跨域？
协议，域名，端口都必须相同，才算在同一个域。也就是说如果协议、域名、端口中有一个不同，就是在不同的域。
<2> 解决跨域有几种方式？
jsonp
是回调函数里的json
cors 
跨域资源共享，这种实现方式需要服务器端的支持，目前所有浏览器均支持，
proxy server
代理服务器，这种方式通过一个代理服务器去请求数据，然后将请求返回的数据再返回给客户端，
服务器之间不存在跨域问题。
<3> json 和 jsonp 数据有什么区别
json 就是数据交换格式
jsonp 是被包含在函数调用中的json, 由2部分组成，回调函数和数据，
回调函数是当响应到来时应该中页面中调用的函数，而数据就是传入回调函数中的json数据
